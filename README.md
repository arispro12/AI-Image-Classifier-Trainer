Custom Image Classifier (PyTorch)

A modern desktop AI image classification trainer built with PyTorch.
Train custom image classes using transfer learning and classify images with a simple GUI.

Custom Image Classifier Screenshot

Features
Train custom image classes
Transfer learning using ResNet18
Real-time training logs
Single image classification
Folder classification support
Progress tracking with ETA
Easy GUI interface
Fast feature extraction
PyTorch-powered backend
Built With
Python 3
PyTorch
Torchvision
Tkinter / CustomTkinter
Pillow
NumPy
